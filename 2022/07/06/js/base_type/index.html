typeof

typeof用于返回以下原始类型

  • 基本类型：number/string/bollean
  • function
  • object
  • undefined

instanceof

instanceof 运算符用于检测构造函数的prototype属性是否出现在某个实例对象的原型链上
也可以理解为是否为某个对象的实例，typeof不能区分数组，但instanceof则可以

截取字符串

使用 slice、substr、substring 函数都可以截取字符串

  • slice substring 第二个参数为截取的结束位置
  • substr 第二个参数指获取字符数量

查找字符串

indexOf

返回某个指定的字符串中首次出现的位置，如果没有找到匹配的字符串，则返回-1

lastIndexOf

从结尾来搜索字符串位置

search

检测字符串中指定的子字符串，或检索与正则相匹配的子字符串，返回对应索引，没有找到匹配的返回-1

includes

判断字符串是否包含子字符串，返回 boolean,第二个参数指查找开始位置

startsWith

检测字符串是否以指定的子字符串开始 返回 boolean

endsWith

检测当前字符串是否是以子字符串结尾的 返回 boolean

替换字符串

replace方法用于字符串的替换操作

使用reduce完成字符串替换关键词链接

replaceAll(regexp|substr, newSubstr|function) 返回一个新字符串，新字符串所有满足 pattern 的部分都已被replacement 替换。pattern可以是一个字符串或一个 RegExp， replacement可以是一个字符串或一个在每次匹配被调用的函数

类型转换

split

split(separator,limit) 把一个字符串分割成字符串数组，不改变原始数组，返回值为数组
separator:可选，字符串或正则表达式，从改参数的指定位置分割
limit:可选，该参数可指定返回的数组的最大长度，如果设置了该参数，返回的子串不会多于这个参数指定的数组

Boolean

布尔类型包括true 与 false两个值，开发中使用较多的数据类型

random

random方法用于返回 >=0 且 <1 的随机数（包括0 但不包括1）

取 2~5 的随机数（不包括 5）公式为：min+Math.floor(Math.random()*(Max-min))
取 2~5 的随机数（包括 5）公式为：min+Math.floor(Math.random()*(Max-min+1))
