类型检测

typeof

typeof用于返回以下原始类型

  • 基本类型：number/string/bollean
  • function
  • object
  • undefined

instanceof

instanceof 运算符用于检测构造函数的prototype属性是否出现在某个实例对象的原型链上
也可以理解为是否为某个对象的实例，typeof不能区分数组，但instanceof则可以

trim删除字符串左右两侧空白
trimLeft删除字符串左侧空白
trimRight删除字符串右侧空白

获取单字符

截取字符串

使用 slice、substr、substring 函数都可以截取字符串

  • slice substring 第二个参数为截取的结束位置
  • substr 第二个参数指获取字符数量

查找字符串

indexOf

返回某个指定的字符串中首次出现的位置，如果没有找到匹配的字符串，则返回-1

lastIndexOf

从结尾来搜索字符串位置

search

检测字符串中指定的子字符串，或检索与正则相匹配的子字符串，返回对应索引，没有找到匹配的返回-1

includes

判断字符串是否包含子字符串，返回 boolean,第二个参数指查找开始位置

startsWith

检测字符串是否以指定的子字符串开始 返回 boolean

endsWith

检测当前字符串是否是以子字符串结尾的 返回 boolean
