探索js函数

发表于2022-07-11|更新于2022-07-11|js
|字数总计:2k|阅读时长:8分钟|阅读量:

函数是将复用的代码块封装起来的模块，在 JS 中函数还有其他语言所不具有的特性

声明方式

匿名函数

函数是对象所以可以通过赋值来指向函数对象的指针，当然指针也可以传递给其他变量，注意后面要以 ;

结束。

  • 函数提升，定义的函数，不能在定义该方法前使用该函数
  • 没有变量定义函数的话存在函数提升，可以在页面任意地方调用

立即执行

立即执行函数指函数定义时立即执行

  • 可以用来定义私有作用域防止污染全局作用域
  • 使用 let/const 有块作用域特性，所以使用以下方式也可以产生私有作用域

参数和argument

形参是在函数声明是设置的参数，实参指在调用函数时传递的指

  • 形参数量大于是实参时，没有传参的形参值为 undefined
  • 实参数量大于形参时，多于的实参将忽略并不报错
  • arguments 是函数获得到所有参数集合

箭头函数

箭头函数时函数声明的简写形式，在使用递归调用，构造函数，事件处理器时不建议使用箭头函数

递归

递归指函数内部调用自身的方式

  • 递归函数必须接收参数（比如我要递归谁）
  • 在递归函数的定义初始，应该有一个判断条件，当参数满足这个条件的时候，函数停止执行，并返回值（指出退出条件，否则就会死循环）
  • 每次递归函数执行自己的时候，都需要把当前参数做某种修改，然后传入下一次递归（每次循环在调用自己一次并传参）
  • 当参数被积累修改到符合初始判断条件了，递归就停止了

回调函数

在某个时刻被其他函数调用的函数称为回调函数

展开语法(点语法)

展开语法或称点语法体现的就是 收/放 特性，做为值时是 放,做为接收变量时是收

this

调用函数时this 会隐式传递给函数指函数调用时的关联对象，也称之为函数的上下文

函数为对象的方法是 this 指向该对象

可以使用多种方式创建对象

构造函数

函数当被 new 时即为构造函数，一般构造函数中包含属性与方法。

  • 构造函数主要用来生成对象，里面的this 默认就是指当前对象

对象字面量

  • 下面的func函数不属于对象方法所以指向window
  • show属于对象方法执行obj对象

箭头函数

箭头函数没有this，也可以理解为箭头函数中的this会继承定义函数时的上下文，可以立即为和外层函数指向同一个this

  • 如果想使用函数定义时的上下文中的this，那就使用箭头函数
  • 事件函数可以理解为对象onclick设置值，所以函数声明时this为当前对象
  • 但使用箭头函数时this为声明函数上下文

apply/call/bind

改变this指针，也可以理解为借用方法

原理分析

构造函数中的this默认就是一个空对象，然后构造函数处理后把这个空对象变得有值

apply/call

call与apply用于显示设置函数的上下文，这两个方法作用一样都是将对象绑定到this，只是在传递参数上有所不同

  • apply用数组传参
  • call需要分别传参
  • 与bind不同call/apply会立即执行函数

bind

bind()是将函数绑定到某个对象，比如a.bind(use)可以理解为将a函数绑定到use对象上即use.a()

  • 与apply/call不同的bind不会立即执行
  • bind是复制函数行为会返回新函数
