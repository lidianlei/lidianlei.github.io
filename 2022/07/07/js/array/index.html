类型转换

可以将数组转换为字符串也可以将其他类型转换为数组

字符串

  • toString()方法
  • String()函数
  • join() 数组方法

Array.from

使用Array.from可将类数组转换为数组，类数组指包含 length 属性或可迭代的对象

展开语法

展开语法以 ... 的形式使用。效果是将一个数组展开放进一个新的数组中，如果在一个字符串前加上展开语法，可以将这个字符串变成同等长度的数组

  • 数组合并,使用展开语法来合并数组相比 concat 要更简单，使用… 可将数组展开为多个值
  • 函数参数,使用展示语法可以替代 arguments 来接收任意数量的参数
  • 节点转换

解构语法

解构是一种更简洁的赋值特性，可以理解为分解一个数据的结构

  • 建议使用 var/let/const 声明

增删

  • push(item1,item2...) 向数组末尾添加一个或多个元素，并返回新数组的长度
  • pop() 删除数组的最后一个元素并返回删除删除的元素
  • unshift(item1,item2) 向数组开头添加一个或多个元素
  • shift() 删除数组的第一个元素并返回删除元素
  • fill(val,start,end) 用于固定值替换数组的元素, val必填 ，填充的值；start可选 ，开始填充位置；end可选，结束填充位置
  • slice(start,end) 从数组中截取部分元素合成新数组 不改变原数组 ，不传第二个参数时截取到数组的最后元素，左闭右开

splice方法

splice(index,howmany,...itemx) 删除，添加 替换数组中的元素 index必须 ；howmany可选，规定应删除多少元素，必须是数字，可以为0；itemx可选，要添加到元素的新元素

清空数组

join

split

concat

copyWidth

使用 copyWidth 从数组中复制一部分到同数组中的另外位置
语法： array.copyWidth(target,start,end)
target：必需，复制到指定目标索引位置
start：可选，元素复制的起始位置
end：可选，停止复制的索引位置（默认为 array.length）.为负值表示倒数

indexOf

indexOf(item,start) 返回数组中可以找到一个给定元素的一个索引，不存在返回 -1

lastIndexOf

lastIndexOf(item,start) 使用 lastIndexOf 从后向前查找元素出现的位置，如果找不到返回 -1

includes

includes(searchElement,fromIndex) 判断一个数组是否包含一个指定的值，返回布尔

find

find 方法找到后会把值返回出来

​ find(function(currentValue,index,arr),thisValue) 返回数组中满足提供的测试函数的第一个元素的值，否则返回 undefined

  • 如果找不到返回值为 undefined
  • 使用includes等不能查找引用类型，因为它们的内存地址是不相等的
  • find 可以方便的查找引用类型

findIndex

findIndex(function(currentValue,index arr),thisValue) 返回数组中满足提供检测函数的第一个元素的索引， 没有返回 -1

reverse

sort

sort 每次使用两个值进行比较 Array.sort((a,b)=>a-b) (a-b 升序 b-a 降序)

  • 返回负数 a 排在 b 前面，从小到大
  • 返回正数 b 排在 a 前面
  • 返回 0 时不动

for

forEach

froeach(function(currentValue,index,arr),thisValue) 用于调用数组的每个元素，并将元素传递给回调函数

for/in

遍历时的 key为数组的索引

for/of

与 for/in 不同的是 for/of 每次循环取其中的值而不是索引。

迭代器方法

keys

values

entries

every

every(function(currentValue,index,arr),thisValue) 用于递归的检测元素，要所有元素操作都符合指定条件才为 true

some

some(function(currentValue,index,arr),thisValue) 用于递归的检测元素，如果有一个返回 true，表达式结果就是真

filter

filter(function(currentValue,index,arr),thisValue) 创建一个新数组，过滤后的

map

map(function(currentValue,index,arr), thisValue) 返回一个新数组，数组中的元素为原始元素调用函数处理后的值

reduce

使用 reduce 与 reduceRight 函数可以迭代数组的所有元素，reduce从前开始 reduceRight 从后面开始

reduce(function(total,currentValue,currentIndex,arr),initialValue) 接受一个函数的累加器，数组中的每个值（从左到右）开始缩减，最终最为一个值

  • total：必需，初始值，或者计算后的返回结果
  • currentValue：必需，当前元素
  • currentIndex：可选，当前元素的索引
  • arr：可选，当前元素所属的数组对象
  • initiaValue：可选，传递给函数的初始值
